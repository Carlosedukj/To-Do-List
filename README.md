# To-Do List

#### Uma aplicação simples e funcional de lista de tarefas desenvolvida com HTML, CSS e JavaScript puro, com foco em prática de lógica, manipulação do DOM e boas práticas de desenvolvimento front-end.

## Funcionalidades
- Adicionar novas tarefas
- Remover tarefas individualmente
- Marcar tarefas como concluídas
- Validação de input com mensagem de erro amigável
- Persistência de dados com LocalStorage
- Manutenção das tarefas mesmo após recarregar a página
- Conceitos aplicados

## Este projeto foi desenvolvido com foco em aprendizado prático dos seguintes conceitos:

-Manipulação do DOM
- Event Delegation (delegação de eventos)
- Estruturação de funções e responsabilidades
- Controle de estado (mesmo sem framework)
- Persistência de dados com localStorage
